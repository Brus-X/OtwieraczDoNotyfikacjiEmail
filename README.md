# Otwieracz Do Notyfikacji Email

Otwieracz: automatyzacja Gmail. Dodawaj powiadomienia z linkiem, wyzwalane przez e-maile. Gdy przyjdzie e-mail od nadawcy X, aplikacja rzuca notyfikację: „Przyszedł e-mail od X, otwórz stronę Y”. Klikasz i od razu otwiera się wskazany URL. Lekka, bezpieczna, działa lokalnie i oszczędza baterię. Idealna do e-dzienników i systemów zamówień.

## Kluczowe cechy
* 🚀 **Szybkość:** Błyskawiczny dostęp do podstron z poziomu notyfikacji.
* 🔒 **Prywatność:** Dane nie opuszczają urządzenia, brak zbierania statystyk.
* 🔋 **Wydajność:** Znikome zużycie energii dzięki systemowej usłudze słuchania.
